# Airbnb Global Performance Dashboard | Power BI

## Project Overview

This project is an interactive Power BI dashboard created to explore Airbnb data from multiple cities. The goal was to understand how listings perform, how customers interact with the platform, and how different factors like ratings, reviews, and host performance vary across locations.

Instead of just building visuals, I focused on creating a dashboard that answers real business questions and presents insights in a clear and interactive way.

## Business Questions

- Which cities have the largest Airbnb market share?
- How have Airbnb listings grown over time?
- Which cities receive the highest customer ratings?
- How do customer reviews change across different seasons?
- What factors contribute to host trust and credibility?

## Dashboard Pages

### 1. Overview Page

Provides a high-level overview of Airbnb performance using key metrics and trends.

**Includes:**
- Listing Growth Over Time
- Room Type Distribution
- Historical Business Trends

---

### 2. Ratings Page

Focuses on market performance, pricing, and customer ratings across different cities.

**Includes:**
- Market Share by City
- Super Host vs Non-Super Host Listings
- Average Price by Room Type
- Overall City Ratings
- Detailed Rating Analysis using Bookmarks

---

### 3. Customer & Trust Analysis

Analyzes customer review behavior and host credibility.

**Includes:**
- Review Frequency Distribution
- Customer Review Patterns
- Trust Factor Analysis
- Host Verification Analysis
- Seasonal Review Trends
